## .env Setup for Vite

1. Create a file named `.env` in the root of your project.
2. Add the following environment variables:
   VITE_NODE_ENV=
   VITE_DEV_IP=
   VITE_PROD_IP=
   VITE_PORT=
   VITE_WS_PORT=
3. Fill in the appropriate values depending on your environment (e.g., localhost or production server).
